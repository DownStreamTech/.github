## About Us

We build **affordable, secure, and innovative digital solutions** powered by Free and Open-Source Software (FOSS). Based in the northeastern region of Cagayan, Philippines, we focus on driving inclusive, sustainable, and technology-enabled development for every community.

### Our Core Pillars

| Affordability | Sustainability | Impact |
|:---:|:---:|:---:|
| Eliminating licensing fees through open-source solutions | Building future-ready, scalable systems | Maximizing value for every peso invested in technology |

## Our Products

| Product | Description |
|---|---|
| **D2AS** | Document Archive & Access System — enterprise-grade document management with full-text search and audit trails |
| **AttendanceHub** | Smart e-Attendance system with QR/barcode integration, IoT sensors, and geolocation verification |
| **iLearn Suite** | Moodle-powered Learning Management System with adaptive learning pathways and interactive content |
| **PeopleCore HRIS** | Human Resource Information System for streamlined workforce management |
| **On-Premise Storage** | NextCloud-powered secure file storage and collaboration |
